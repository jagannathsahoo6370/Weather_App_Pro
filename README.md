# 🌦 Weather App Pro

A modern weather application built using **Python** and **PyQt6** that provides real-time weather information for any city using the OpenWeatherMap API.

---

## 🚀 Features

- Search weather by city name
- Real-time temperature display
- Weather condition information
- Feels-like temperature
- Humidity percentage
- Wind speed
- Atmospheric pressure
- Sunrise and sunset timings
- Modern dark-themed user interface
- User-friendly design

---

## 🛠 Technologies Used

- Python
- PyQt6
- Requests
- OpenWeatherMap API

## 📦 Installation

### Clone the repository

git clone https://github.com/jagannathsahoo6370/weather-app-pro.git

### Navigate to the project folder

cd weather-app-pro

### Install dependencies

pip install -r requirements.txt

### Run the application

python weather_app.py
